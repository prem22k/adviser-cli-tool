# Contributing to Adviser-CLI

Thank you for your interest in contributing! Adviser-CLI is a community-driven project aimed at building the best local-first zero-infrastructure intelligence hub.

## Development Setup

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/prem22k/adviser-cli-tool.git
    cd adviser-cli-tool
    ```

2.  **Create a virtual environment:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -e ".[dev]"
    ```

## Contribution Workflow

1.  **Fork the repository.**
2.  **Create a feature branch:** `git checkout -b feature/awesome-feature`
3.  **Make your changes.**
4.  **Run tests (if applicable).**
5.  **Submit a Pull Request.**

## Code Style
We use standard code styles for Python. Please ensure your files pass verification checks and import perfectly before submitting.
