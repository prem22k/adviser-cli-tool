# Adviser: Project Vision & Architecture

## The Core Identity
**"The RAG app for people who don't want an app."**
Adviser is the terminal-native **Global Intelligence Hub** for the private professional. It isn't just a RAG chat; it is an agentic layer that bridges local documents with external workflows (Slack/GitHub/Jira) while maintaining absolute privacy. It transitions the user from "searching for info" to "validated execution" via automated sandboxing and actionable context. Adviser is engineered to run flawlessly on an 8GB hardware baseline while dynamically scaling to leverage high-end gaming rigs and GPUs, putting the user in total control of their data, hardware, and API costs.

# Core Production-Grade Engine Specifications

## 1. The "Bring-Your-Own-Pipeline" (BYOP) Engine
Adviser makes zero assumptions about your models. The source of truth is a portable YAML configuration file, generated by an interactive CLI setup wizard. 
* **Express Setup:** Auto-detects system RAM, suggests optimized defaults, configures a primary API key, and gets you chatting with your documents in 60 seconds.
* **Developer Mode:** Grants absolute control over chunk sizes, overlap, generation parameters, and custom routing rules.
* **Hardware-Agnostic Scaling:** Users can plug in model IDs for local HuggingFace models, Ollama, or cloud APIs. 
* **AirLLM Integration:** For maximum privacy and quality on constrained hardware, Adviser supports AirLLM, allowing massive 70B+ parameter models to run locally by loading layers directly from disk.

## 2. Intelligent Routing & Circuit Breaking
A robust, session-aware routing system designed for production-level reliability and cost management.
* **Priority Cascades:** Define a fallback order for both embedding and generation (e.g., 1. Free API, 2. Fast Local Model, 3. AirLLM Heavy Model).
* **Smart Failover:** If a primary provider hits a rate limit (HTTP 429) or a timeout (HTTP 503), the pipeline automatically and seamlessly routes the query to the next provider.
* **Session Memory:** The system acts as a local circuit breaker. If an API is locked out during a chat session, Adviser remembers and skips it for subsequent prompts, preventing wasted time and terminal hanging.

## 3. Dynamic Profile Management
Adviser natively supports context switching without cross-contaminating your vector data or system prompts.
* **Total Isolation:** Create named environments for entirely different domains (e.g., adviser --profile finance vs. adviser --profile engineering).
* **Independent Rulesets:** Each profile is linked to its own isolated ChromaDB collection, its own system prompt, and its own dedicated routing configuration.

## 4. Advanced Ingestion & Layout-Aware VisionRAG
Built to handle real-world utility beyond simple markdown files.
* **Universal Formats:** Full baseline support for parsing standard text, PDFs, and DOCX files.
* **VisionRAG (ColPali):** For complex documents where standard text extraction destroys tables and charts, Adviser offers a visual indexing mode. It treats PDF pages as images, utilizing Vision-Language Models to embed layouts and visual data accurately by projecting ColPali `vidore/colpali-v1.2` patch embeddings into high-fidelity cosine indexes.
* **Optimized Processing:** Batched embeddings and concurrent I/O reading drastically speed up the ingestion of massive repositories, paired with rich terminal progress bars.

## 5. The Digest Engine
The powerful Map-Reduce summarizer is elevated to a primary user tool for corpus-wide analysis.
* **Batch Processing:** Running adviser digest triggers a comprehensive extraction job over an active profile's entire loaded corpus, perfect for generating master summaries or timelines.
* **Plan Mode:** Before firing off hundreds of API requests, the engine reads the database and outputs a terminal summary estimating time and potential API costs, requiring user confirmation to proceed safely.

## 6. Model Context Protocol (MCP) Integration
Adviser is not just a standalone terminal chat—it is the local brain for your broader development environment.
* **IDE Handoff:** Launching with adviser --mcp turns your local RAG database into an active context server. 
* **Agentic Synergy:** Modern AI coding tools (like Cursor, Claude Code, or Windsurf) can securely query your Adviser profiles to write code based on your local, private documentation and notes.
